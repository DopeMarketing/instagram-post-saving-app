# Instagram post saving app

## Overview
A simple mobile app that allows users to save Instagram posts they like and organize them for easy reference and reuse of content ideas. Users can bookmark posts, categorize them, and revisit saved content when planning their own Instagram strategy.

## Problem
Instagram users lose track of posts they want to reference later, have no way to organize saved content by themes or categories, and struggle to find inspiration when creating new posts

## Target User
Content creators, social media managers, and Instagram users who want to organize and reference inspiring posts for their own content creation

## Features
- Save Instagram posts via URL or screenshot
- Organize saved posts into custom categories/folders
- Search and filter saved posts
- Add personal notes to saved posts
- Export saved posts as image collections
